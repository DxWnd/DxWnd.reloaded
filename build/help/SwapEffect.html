Forces Swap Effect

FORCESWAPEFFECT


This experimental flag forces the selected SwapEffect mode during the CreateDevice/Reset methods. In some cases, it may improve compatibility or performance. You can choose from the following modes:


  • Discard
  • Flip
  • Copy
  • Overlay
  • FlipEx
